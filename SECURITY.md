# Reporting a Vulnerability

If you discover a security issue in Bubbly Maps, please **do not open a public issue**.  
Instead, contact us privately at [security@myapp.com](mailto://security@myapp.com).

In your report, please include:  
- A description of the vulnerability
- Steps to reproduce
- Any potential impacts

We will respond promptly and work with you to resolve the issue.

## Supported Versions

Only the latest releases are actively maintained for security fixes.  
Older versions may not receive updates.
